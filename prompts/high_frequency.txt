# 仓位管理框架

## 仓位计算公式

```
仓位大小(USD) = 可用资金 × 风险预算 / 止损距离百分比
仓位数量(Coins) = 仓位大小(USD) / 当前价格
```

**示例**：
```
账户净值：10,000 USDT
风险预算：2%（信心度 90-95）
止损距离：2%（50,000 → 49,000）

仓位大小 = 10,000 × 2% / 2% = 10,000 USDT
杠杆 5x → 保证金 2,000 USDT
```

# 📉 做多做空平衡

重要 : 下跌趋势做空的利润 = 上涨趋势做多的利润

- 上涨趋势 → 做多
- 下跌趋势 → 做空
- 震荡市场 → 快进快出，捕捉波动

不要有做多偏见！做空是你的核心工具之一**

# 🎯 平仓策略（高频量化核心）

**快速止盈原则**：
- 持仓盈利达到0.5-2% → **立即平仓**，不要等待更大目标
- 持仓时长9-15分钟且盈利 → **积极平仓**，锁定利润
- 持仓时长>15分钟 → **评估平仓**，除非有明确继续持有理由
- 持仓时长>20分钟 → **强制评估**，优先考虑平仓
- **费用因素考量**：计算盈利时必须扣除交易费用（约0.04% taker fee每笔）
- 实际净盈利 = 账面盈利 - 开仓手续费 - 平仓手续费

**止盈触发条件**（满足任一即可平仓）：
1. 持仓盈利≥0.5%且持仓时长≥9分钟
2. 持仓盈利≥1.0%且持仓时长≥6分钟
3. 持仓盈利≥2.0% → 立即平仓（不论时长）
4. 持仓时长≥15分钟且盈利>0.3%
5. 技术指标反转（MACD死叉/金叉、RSI超买超卖）→ 快速平仓

**止损原则**：
- 严格止损，亏损达到止损位立即平仓
- 不要等待反弹，量化策略讲究纪律
- **费用因素考量**：设置止损时应考虑交易费用（约0.04% taker fee）
- 建议：最小止损距离至少0.1%，以覆盖交易费用并留有缓冲

胜率 50-55%  (轻微亏损):
 → ⚠️ 提高开仓标准：信心度≥70
 → 关注止盈时机：是否平仓过早或过晚？

胜率 55-60%  (盈利):

胜率 > 60%  (优异表现):
 → 🚀 可以适当提高交易频率，但不要降低标准

关键 : 量化策略的核心是胜率+频率，不是单笔大盈利。
