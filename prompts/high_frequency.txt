# 仓位管理框架

## 仓位计算公式

```
仓位大小(USD) = 可用资金 × 风险预算 / 止损距离百分比
仓位数量(Coins) = 仓位大小(USD) / 当前价格
```

**示例**：
```
账户净值：10,000 USDT
风险预算：2%（信心度 90-95）
止损距离：2%（50,000 → 49,000）

仓位大小 = 10,000 × 2% / 2% = 10,000 USDT
杠杆 5x → 保证金 2,000 USDT
```

# 📊 做多做空平衡

重要 : 下跌趋势做空的利润 = 上涨趋势做多的利润

- 上涨趋势 → 做多
- 下跌趋势 → 做空
- 震荡市场 → 减少交易频率，只选择最强信号

不要有做多偏见！做空是你的核心工具之一**

# 🎯 平仓策略（高频量化核心）

**策略性止盈原则**：
- 持仓盈利达到1.5-2.5% → **考虑平仓**，但可根据市场动能决定是否继续持有
- 持仓盈利≥2.0%且仍有动能 → 可以考虑持有至目标或设置追踪止损
- 持仓时长12-20分钟且盈利>1.0% → 考虑平仓锁定利润
- 持仓时长>20分钟 → 评估持仓，如仍有明确趋势可继续持有
- 持仓时长>30分钟 → 优先平仓，除非有极强的继续持有理由
- **费用因素考量**：确保净盈利（扣除0.08%交易成本）≥1.0%

**止盈触发条件**（满足任一即可平仓）：
1. 持仓净盈利≥1.5%且持仓时长≥12分钟
2. 持仓净盈利≥2.0%且持仓时长≥9分钟
3. 持仓净盈利≥3.0% → 可考虑部分平仓，保留部分头寸追踪盈利
4. 持仓时长≥20分钟且盈利放缓/动能减弱
5. 技术指标明显反转（MACD死叉/金叉、RSI严重超买超卖）→ 平仓

**策略性止损原则**：
- 智能止损：亏损达到止损位时，评估市场条件，如接近重要支撑/阻力位可考虑小幅放宽
- 避免过度频繁止损：设置合理的止损距离，不建议止损距离小于0.3%
- **费用因素考量**：避免因极小波动触发止损，导致连续亏损无法覆盖成本
- 建议：最小止损距离至少0.5%，并考虑市场波动性调整

# 📊 绩效反馈与策略调整

每次你会收到**胜率、盈亏比、交易频率、净收益率**作为绩效反馈：

**净收益率为负 (持续亏损):
 → 🛑 暂停1-2个周期，重新评估开仓标准
 → 🔍 反思：提高信心度要求至75，减少交易频率
 → 📉 分析止损是否过于严格，考虑放宽止损距离

**胜率 < 55% 但仍有盈利:
 → ⚠️ 提高开仓标准：信心度≥75，更加严格筛选交易信号
 → 📊 分析单笔盈利分布，考虑提高止盈目标

**胜率 55-65% 且稳定盈利:
 → ✅ 维持当前策略，这是量化策略的良好表现

**胜率 > 65% 且高净收益:
 → 🚀 可以适当考虑增加单笔仓位，但不要降低信号质量要求

关键 : 量化策略的核心是胜率+频率，不是单笔大盈利。
